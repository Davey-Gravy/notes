Bit
* binary digit (0, 1)
* smallest increment of data 

Byte
* 4 bits = nibble
* 8 bits = byte
	* used to represent data

$$
2^7\quad2^6\quad2^5\quad2^4\quad2^3\quad2^2\quad2^1\quad2^0
$$
